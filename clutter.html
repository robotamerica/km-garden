tool shed
machines 💻
  • macbook air m1 — mobile command node. writing, coordination, travel. macos, yabai, iterm2, zsh, homebrew.
  • hp t740 thin client — anchored workstation. arch linux + hyde, hyprland, kitty, waybar. systems lab, gis, visual + audio experiments.
  • android (honor) — pocket terminal. aio launcher + termux. ambient computing, capture, ssh into arch box from anywhere.
digital toolchain ⛓️
  • editors: neovim (main · ovipoet · dotmd), vs code
  • shell: zsh · starship · fzf · ripgrep · fd · bat · jq · htop
  • writing + notes: dotmd.nvim (zettelkasten) · bear blog · markdown-first
  • web: html · css · vanilla js · leaflet.js · bash scripting
  • gis + mapping: qgis · ogr2ogr · mapshaper · geojson · leaflet
  • media: mpv · yt-dlp · imagemagick · ffmpeg · pandoc · lightroom · gimp
  • sound: strudel (generative) · boom (mpv + yt-dlp wrapper)
  • network: tailscale · ssh · copyparty
  • termux stack: openbook (epy + fzf library) · newsboat · zettlemind · boom · termux-games
  • deploy: github pages · surge · netlify · cloudflare r2
  • version control: git · github (robotamerica)

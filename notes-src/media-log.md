a static daily media log. entries cover reading, music, and video. a cli script appends entries, updates a json index, and pushes to github pages. no database, no account, no platform holding the data.

media-log is a [[carrier-bag]] tool: it gathers and holds without demanding resolution. a record of what passed through, kept in plain json i can read anywhere.

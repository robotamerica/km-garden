a personal music cli built on mpv, yt-dlp, and fzf. searches youtube and bandcamp, keeps a listening history, and plays audio without a browser or account. no server, no tracking, no update nag.

boom is a permatechnic tool: it uses things that already exist, composted into something small and personal. a [[zenops]] artifact. the history file is a plain tsv i can read in any text editor.

source at <a href="https://github.com/robotamerica" target="_blank" rel="noopener">github.com/robotamerica</a>

a minimalist bbs-inspired web architecture and quiet alternative to feeds and timelines. a grove is a local index of linked html pages called seeds. each seed is a standalone html file with no backend, no algorithm, and no timeline.

concepts: commons (philosophy and network), grove (index), seed (single html page), and sprout (act of publishing). seeds are submitted by email. random seed navigation via js and a seeds.json file.

404 commons is a small architectural expression of le guin's political imagination: no algorithm, no hierarchy, no feed. a grove of html pages linked by care.

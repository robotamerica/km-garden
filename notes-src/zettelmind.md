a personalised fork of <a href="https://neovimcraft.com/plugin/y3owk1n/dotmd.nvim/" target="_blank" rel="noopener">dotmd.nvim</a> that lives across my devices and launches via a neovim alias.

it features a rainbow dashboard, journal, todos, inbox, work notes, telescope search, tag index, git sync, graph view, zettel notes, and a garden publish command for [[rhizome-note]] notes.

built on top of dotmd.nvim with custom plugins for work, tags, and git workflows. the ops expression of [[zenops]] applied to knowledge work.

source at <a href="https://github.com/robotamerica/zettelmind" target="_blank" rel="noopener">github.com/robotamerica/zettelmind</a>
